## Structure

    .
    ├── data/                         # generated by scraper 
    |   ├── images/                   # scraped directory of images by id
    |   ├── <category>.csv            # category product data
    |   └── <category>-links.csv      # links of all category in product page
    └── scraper.py

### Scraped Data

Each product includes
* Product ID (specified by Pandora)
* Category
* URL
* Name
* Description
* Price
* Dimensions
* Materials

## Installation

```
pip install -r requirements
playwright install
```

## Usage
```
python3 scraper.py {url | path to link file}
```

Given a category url scraper will collect all product links from the page and save them to <category>-links.csv. Visit each product page and scrape and append results to <category>.csv. If given a path to a pre-existing csv of links it will skip the product extraction (note links csv must have a - in the file name to avoid overwrite).

### Example

```
python3 scraper.py https://us.pandora.net/en/bracelets
```
